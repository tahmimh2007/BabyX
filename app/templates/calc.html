📚 AP Calculus BC Study Guide

📦 Unit 1: Limits and Continuity

  • Limit Definition: \(\lim_{x \to c} f(x) = L\) means f(x) approaches L as x approaches c.
  • One-sided Limits: Left: \(\lim_{x \to c^-}\), Right: \(\lim_{x \to c^+}\)
  • Continuity: f is continuous at c if: f(c) exists, limit exists, and they are equal.
  • Intermediate Value Theorem (IVT): If f is continuous on [a, b], and N is between f(a) and f(b), then ∃ c in (a, b) s.t. f(c) = N.

📦 Unit 2: Differentiation: Definition and Fundamental Properties

f'(x) = lim(h→0) [f(x + h) - f(x)] / h
  • Power Rule: d/dx[xⁿ] = nxⁿ⁻¹
  • Product Rule: (uv)' = u'v + uv'
  • Quotient Rule: (u/v)' = (v u' - u v') / v²
  • Chain Rule: dy/dx = dy/du × du/dx

📦 Unit 3: Composite, Implicit, and Inverse Functions

  • Implicit Differentiation: Differentiate y using Chain Rule, then solve for dy/dx.
  • Inverse Derivative: \(\frac{d}{dx}[f^{-1}(x)] = \frac{1}{f'(f^{-1}(x))}\)

📦 Unit 4: Applications of Derivatives

  • Related Rates: Differentiate both sides wrt t.
  • Motion: s(t): position, v(t) = s'(t), a(t) = v'(t)
  • Optimization: Critical points via f'(x)=0, test for max/min.
  • Linear Approximation: f(x) ≈ f(a) + f'(a)(x − a)

📦 Unit 5: Analytical Applications of Derivatives

  • 1st Derivative Test: Sign change → max/min
  • 2nd Derivative Test: f''(c) > 0 → min; f''(c) < 0 → max
  • Mean Value Theorem: f'(c) = (f(b) − f(a)) / (b − a)

📦 Unit 6: Integration and Accumulation of Change

  • FTC Part 1: d/dx[∫ₐˣ f(t)dt] = f(x)
  • FTC Part 2: ∫ₐᵇ f(x)dx = F(b) − F(a)
  • Average Value: (1 / (b − a)) ∫ₐᵇ f(x)dx

📦 Unit 7: Differential Equations

  • Separable: dy/dx = g(x)h(y) → integrate both sides
  • Logistic: dP/dt = rP(1 − P/K)

📦 Unit 8: Applications of Integration

  • Area Between Curves: ∫ₐᵇ [f(x) − g(x)] dx
  • Volumes: Disk: π∫ₐᵇ [R(x)]² dx
  • Shell: 2π∫ₐᵇ (radius)(height) dx

📦 Unit 9: Parametric, Polar, Vector

  • Parametric dy/dx: dy/dx = (dy/dt) / (dx/dt)
  • Polar Area: (1/2) ∫ₐᵇ r² dθ

📦 Unit 10: Infinite Sequences and Series

  • Tests: nth-Term, Geometric, p-Series, Ratio
  • Taylor Series: f(x) = Σ f⁽ⁿ⁾(a)/n! × (x−a)ⁿ
  • Common: eˣ, sin x, cos x, ln(1+x)
