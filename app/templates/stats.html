📚 AP Statistics Study Guide (Units 6–9)

📖 Unit 6: Confidence Intervals (CIs)

🔹 What is a Confidence Interval?

A confidence interval is a range of plausible values for a population parameter, based on a sample statistic.

Point Estimate ± (Critical Value) × (Standard Error)

🔹 Confidence Level

95% CI: If we repeated this method many times, about 95% of the intervals would capture the true parameter.

🔹 Conditions for Inference

  • Random: Data from random sample or random assignment
  • Normal: For proportions: np ≥ 10 and n(1-p) ≥ 10; for means: population normal or n ≥ 30
  • Independence: 10% condition if sampling without replacement

🔹 Formulas

One-Proportion z-Interval

p̂ ± z* √( p̂(1 - p̂) / n )

One-Sample t-Interval

x̄ ± t* (s / √n)

Two-Sample Proportions

(p̂₁ - p̂₂) ± z* √( p̂₁(1 - p̂₁)/n₁ + p̂₂(1 - p̂₂)/n₂ )

Two-Sample Means (unpaired)

(x̄₁ - x̄₂) ± t* √( s₁²/n₁ + s₂²/n₂ )

📖 Unit 7: Significance Tests

🔹 Steps in a Hypothesis Test

  1. State Hypotheses: H₀ and Hₐ
  2. Check Conditions
  3. Calculate Test Statistic
  4. Find p-value
  5. Make a Decision

🔹 Test Statistics

One-Proportion z-Test

z = (p̂ - p₀) / √( p₀(1 - p₀) / n )

One-Sample t-Test

t = (x̄ - μ₀) / (s / √n)

Two-Proportion z-Test

z = (p̂₁ - p̂₂) / √(p_c(1 - p_c)(1/n₁ + 1/n₂))

where p_c = combined proportion

Two-Sample t-Test

t = (x̄₁ - x̄₂) / √(s₁²/n₁ + s₂²/n₂)

🔹 Errors

  • Type I: Rejecting true H₀
  • Type II: Failing to reject false H₀
  • Power: Probability of rejecting a false null

📖 Unit 8: Inference for Categorical Data (Chi-Square Tests)

🔹 Types of Chi-Square Tests

  • Goodness-of-Fit: One variable, one population
  • Homogeneity: One variable, multiple populations
  • Independence: Two variables, one population

🔹 Test Statistic

χ² = Σ (O - E)² / E

df =

  • GOF: k - 1
  • Homogeneity or Independence: (r - 1)(c - 1)

📖 Unit 9: Inference for Quantitative Data (Regression)

🔹 Slope Inference

Testing for linear association between x and y

t = (b - 0) / SE_b

df = n - 2

🔹 Conditions

  • Linear relationship (scatterplot)
  • Independent observations
  • Normality of residuals
  • Equal spread in residuals (homoscedasticity)

🔹 Confidence Interval for Slope

b ± t* × SE_b

✅ Final Exam Pro Tips

  • Always state parameters in context
  • Check and state all conditions
  • Use complete sentences when interpreting results
  • Know when to use z vs. t vs. χ²
